# Мини-игры на JavaScript 🎮

Сборник из 6 интерактивных браузерных игр. Проект демонстрирует навыки работы с логикой JavaScript, манипуляции DOM-деревом и создания адаптивных интерфейсов.

[**Посмотреть проект вживую**](https://vovamikh088.github.io/Kursovay/)

<img src="img/preview.png" width="400" alt="Превью проекта">

### Технологии
* **Vanilla JavaScript** — вся игровая логика без использования фреймворков.
* **HTML5 / CSS3** — семантическая и современная верстка.
* **SCSS** — модульная структура стилей и адаптивные стили.
* **БЭМ** — четкая архитектура классов для удобной поддержки.

### Список игр
1. **Угадай число** — работа со случайными числами и условиями.
2. **Простая арифметика** — генерация примеров разной сложности.
3. **Переверни текст** — практика работы со строками и методами массивов.
4. **Камень, ножницы, бумага** — классическая игра против ИИ.
5. **Простая викторина** — работа с объектами данных и проверкой ответов.
6. **Генератор случайных цветов** — динамическое управление стилями страницы.

### Структура проекта
* `index.html`, `index.js` — основа приложения и логика.
* `style.scss` — главный файл стилей.
* `_adaptive.scss`, `_mibile.scss` — модули для адаптивности в корне.
* `style.css` — скомпилированный файл.
* `img/` — папка с изображениями и декоративной графикой.

---

### Как запустить локально
Проект не требует дополнительных настроек. Просто откройте `index.html` в браузере или запустите через локальный сервер:

```bash
# С помощью Node.js (если установлен)
npx serve

# С помощью Python
python -m http.server 8000
```
